## 🚀 Как запустить (для Янислава)

### 5 шагов:
1. **Telegram бот:** @BotFather → `/newbot` → сохранить токен
2. **Google API:** Cloud Console → создать ключ для Sheets
3. **Render.com:** загрузить `render.yaml` через Blueprint
4. **Переменные:** добавить токены в Render
5. **Проверка:** `/start` боту → должно работать
