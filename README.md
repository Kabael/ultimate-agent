## 🚀 Быстрый старт (5 минут)

### 1. Подготовка аккаунтов
1. **Render.com** - зарегистрируйтесь через GitHub (бесплатно)
2. **Telegram** - создайте бота через @BotFather:
   ```

   ```
   Сохраните токен: `1234567890:ABCdefGHIjklMNOpqrsTUVwxyz`
3. **Google Cloud Console** - получите API ключ для Google Sheets

### 2. Развёртывание на Render.com
1. Зайдите на [render.com](https://render.com)
2. Нажмите "New +" → "Blueprint"
3. Загрузите файл `render.yaml` из этого архива
4. Нажмите "Apply" - все сервисы создадутся автоматически

### 3. Настройка переменных окружения
После создания сервисов добавьте:

**В ultimate-agent-api:**
- `TELEGRAM_BOT_TOKEN` = токен вашего бота
- `GOOGLE_SHEETS_API_KEY` = ваш Google API ключ
- `DEEPSEEK_API_KEY` = `sk-a5f795364f2c4b04b01f9f2909d516e2`

**В openclaw-gateway:**
- `DEEPSEEK_API_KEY` = `sk-a5f795364f2c4b04b01f9f2909d516e2`

### 4. Проверка работы
1. Откройте `https://ultimate-agent-api.onrender.com/health` - должно быть "OK"
2. Напишите `/start` вашему боту в Telegram
3. Бот должен ответить приветствием

## 🏗️ Архитектура системы

### Сервисы (все на Render Free Tier):
1. **ultimate-agent-api** (Node.js, порт 3000) - основной бэкенд
2. **openclaw-gateway** (Docker) - AI оркестратор
3. **n8n** (Docker) - workflow автомация
4. **duckduckgo-proxy** (Docker) - поиск без API ключа
5. **ultimate-agent-db** (PostgreSQL) - база данных

### API Endpoints:
- `POST /api/create-client` - регистрация нового клиента
- `GET /api/client/:id` - информация о клиенте
- `POST /api/webhook/n8n` - вебхук от n8n
- `GET /health` - проверка здоровья

## 🤖 Telegram Бот

### Команды:
- `/start` - регистрация клиента
- `/articles [артикулы]` - указать артикулы для мониторинга
- `/status` - текущий статус
- `/help` - помощь

### Работа с клиентами:
1. Клиент пишет `/start`
2. Бот создаёт Google Sheets таблицу
3. Сохраняет chat_id в базу
4. Отправляет ссылку на таблицу
5. Активирует n8n workflow для мониторинга

## 🛠️ Устранение неполадок

### Если бот не отвечает:
1. Проверьте `TELEGRAM_BOT_TOKEN` в Render
2. Убедитесь, что бот запущен: `@BotFather` → `/mybots`
3. Проверьте логи: Render → ultimate-agent-api → Logs

### Если Google Sheets не создаются:
1. Проверьте `GOOGLE_SHEETS_API_KEY`
2. Убедитесь, что API включён в Google Cloud Console
3. Проверьте квоты API

### Если n8n не работает:
1. Откройте `https://n8n.onrender.com`
2. Проверьте импортирован ли workflow
3. Проверьте вебхуки в настройках
